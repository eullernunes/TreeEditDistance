Tree Edit Distance Project
Descrição
Este projeto implementa e compara dois algoritmos para calcular a distância de edição entre árvores: o algoritmo de Zhang e Shasha (1989) e o algoritmo de Philip N. Klein. A distância de edição de árvores (Tree Edit Distance - TED) é uma métrica que quantifica o esforço necessário para transformar uma árvore em outra através de operações de inserção, deleção e substituição de nós.
